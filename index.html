Testsuite

Testsuite is a small framework to write unit tests based on javascript. It has a quite simple features and its not made to compete with any other similar frameworks such as Jasmine, Mocha, etc. It's a study case project.

Download »

Modules

Here's a quick module example:

              
Testsuite.module('My module description', function (module) {
  // thats my module code
});

Every module you give you a module argument that enables you to write your tests.

Test

Writing tests would be like::

              
Testsuite.module('My module description', function (module) {
  module.test('My module test', function(assert) {
    // my test assertions
  });
});

Every test gives you a assert argument that enables you to write your asserts.

Assert

Writing asserts would be like::

              
Testsuite.module('My module description', function (module) {
  module.test('My module test', function(assert) {
    assert.equals(1, 1);
  });
});

And finaly every test gives you a assert argument that enables you to write your asserts.

Available asserts

Here's some asserts to use in your tests

equals

Assert values to equals

              
assert.equals(1, 1); // true
assert.equals(1, '1'); // false
assert.equals(2, 1); // false

notEquals

Assert values not to equals

              
assert.notEquals(1, 1); // false
assert.notEquals(1, '1'); // true
assert.notEquals(2, 1); // true

toContain

Assert that values contains in array

              
var a = [1, 2, 3];
assert.toContain(a, 1); // true
assert.toContain(a, '1'); // false
assert.toContain(a, 4); // false

notToContain

Assert that values not contains in array

              
var a = [1, 2, 3];
assert.notToContain(a, 1); // false
assert.notToContain(a, '1'); // true
assert.notToContain(a, 4); // true
